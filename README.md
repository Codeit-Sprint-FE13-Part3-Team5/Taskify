# Taskify

코드잇 스프린트 13기 Part 3 과정에서 5팀이 진행한 중급 프로젝트 **Taskify** 레포지토리입니다.

## ✅ 컨벤션

### 타입

- **feat** : 새로운 기능 추가
- **fix** : 버그 수정
- **docs** : 문서 내용 변경
- **style** : 코드 스타일 변경(코드 포메팅, 코드 변경이 없는 경우)
- **design** : 사용자 UI 디자인 변경(CSS 등)
- **refactor** : 코드 리팩토링
- **test** : 테스트 코드 작성
- **build** : 빌드 파일 수정
- **ci** : CI 설정 파일 수정
- **perf** : 성능 개선
- **chore** : 빌드 수정, 패키지 매니저 설정, 운영 코드 변경이 없는 경우
- **rename** : 파일명 혹은 폴더명을 수정한 경우
- **remove** : 파일 삭제만 한 경우

### 커밋 메세지

